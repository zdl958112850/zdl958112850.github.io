穷人无法摆脱穷 #

摆脱贫穷难, 因为是陷阱, 生下来就设定好的陷阱, 让人舒服, 让人意识不到, 穷了就想着省钱, 越省钱越没钱, 穷了就想稳定, 越稳定越抓不住机会

想摆脱出去的方法: 反人性 舒服的, 习惯的, 享受的就不要做, 在你的舒适区. 因为它在你的习惯内, 你就意识不到, 不能摆脱 六点: 最重要最后一点

  1. 越穷越爱面子, 买贵的东西充门面, 把面子放到一边, 不要为面子付费, 只有你知道, 你的目标是什么, 你可以付出多大代价, 你可以为了目标放弃什么
  2. 投资和消费是相互矛盾的, 消费越多, 留给投资的越少, 越是吃喝玩乐, 越没钱投资自己, 就越逃不出这个陷阱, 越穷就要克制这种欲望把消费压缩到最低, 才有钱投资, 用在未来.
  3. 舍得为知识付费, 投资不是理财而是投资自己, 最好的回报是知识, 把吃喝玩乐省下的钱用在买书, 学习方面
  4. 该踩坑就踩坑, 穷的时候不踩坑, 后面踩坑付出的代价就更大, 吃亏越早试错成本就越小
  5. 寻找不确定性, 越穷越安于现状, 越没法突破
  6. 切换环境去更大的城市, 去顶尖人才最多的地方. 去想去但从来没去的地方.去竞争最激烈的地方.

结婚不要去买钴戒 #

钻戒流通性差, 不是投资理财产品, 购买会伤害本金.

不要对知识功利 #

要广泛摄取, 最终你会发现, 以前随意摄取的知识都可以融汇一起.

时间管理 #

  1. 当你不需要用到大脑的时候，你才需要时间管理
  2. 上升到统筹环节时才发现时间不够用。
  3. 人生最大权重是第零步，权重，策略，方向，取舍。
  4. 时间管理的真正关键是找出最大权重，然后不计成本的投入，把时间放在决策上。只做最重要的那一件事儿。
  5. 时间不靠管理，靠创造
