旋转坐标轴理解 #

  • x轴就是水平向右的轴
  • y轴就是水平向上的轴
  • z轴就是垂直屏幕向外的轴

移动 #

  • transform: translateX(100px) 向水平方向移动100px, 如果是百分比,是相对于元素的宽高来进行比较的 注意: x和y是可以有百分比的, 但是z轴是没有的

z轴移动 #

  • transform: perspective(900px); 开启透视

透视 #

  • 可以单独给每个元素设置, 也可以给父盒子整体设置, 整体观察和单独观察会有差异
    • 单独观察
    • 整体观察
  • 两种设置方法
    • 函数 transform: perspective(600px) 会对整个生效
    • 规则perspective: 600px 里面的旋转不一致
  • perspective-origin: left bottom 调整观看的角度
  • backface-visibility: hidden 控制背面不可见

案例: 实现卡片翻转效果 #

效果: 

<!DOCTYPE html>
